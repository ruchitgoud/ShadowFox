<h5>Project 1 — AI Autocorrect Keyboard using N-Gram Language Model</h5>

Overview

Developed a beginner-friendly AI autocorrect keyboard that predicts the next word using an N-Gram language model trained on a text dataset.

Key Features
Next-word prediction
Text preprocessing
N-Gram language modeling
Context-based word suggestions

<h5>Project 2 — Loan Approval Prediction using Machine Learning</h5>

Overview

Built a machine learning model to predict whether a loan application should be approved based on applicant information such as income, credit history, education, employment status, and other financial factors.

Key Features
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Feature encoding
Model training and evaluation
Loan approval prediction
